# Blog Preview Card

This project involved creating a blog preview card using HTML and CSS. The task was to replicate a modern article card design featuring a featured image, category tag, publication date, headline, brief summary, and author information.

## Project Summary

The purpose of this project was to strengthen fundamental frontend development skills by building a visually appealing . The design includes a vibrant pink background, a centered card layout, a bold shadow effect, and subtle hover and focus effert to enhance user interaction.

## Key Takeaways

Through this project, I gained hands-on experience with structuring web content using semantic HTML and styling components with CSS. I improved my understanding of layout techniques, particularly using Flexbox to align and center elements effectively. I also practiced applying spacing through padding and margins, as well as enhancing the overall appearance with borders, rounded corners, shadows, and interactive hover states.
